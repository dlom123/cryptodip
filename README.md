# CryptoDip

## Project setup
1. Install dependencies

```
npm install
```

2. Build `dist` directory (front-end)
```
npm run build
```

3. Run both the API and front-end with a single command
```
# development (hot-reload using nodemon)...
npm run dev

# ...or production
npm run server
```

### Lints and fixes files
